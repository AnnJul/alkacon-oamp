<![CDATA[Alkacon OpenCms Add-On Module Package (OAMP) ]]> The Alkacon OpenCms Add-On Module Package (also called OAMP) is a set of free, open source extension modules for OpenCms. Alkacon OAMP adds front-end related features to OpenCms that may be useful in case special functionalities are required.

The Alkacon OAMP modules are made available free of charge as open source from Alkacon Software GmbH, the company also responsible for most of the development of the open source core OpenCms system. Please click here for the OAMP license terms.

The Alkacon OAMP modules are installed on a standard OpenCms system. The source code for the OAMP modules is available in the public OpenCms CVS tree. All modules are distributed separately.
