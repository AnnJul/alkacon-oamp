1. Module functions

With the web optimization module, it is possible to compress js and css files 
on the fly to reduce your webpage bandwidth, as well as to merge several
files together to reduce the amount of requests.
It also allows you to create css sprites on-the-fly to reduce the amount of requests.

Which are essential features for optimizing modern webpages performance.
