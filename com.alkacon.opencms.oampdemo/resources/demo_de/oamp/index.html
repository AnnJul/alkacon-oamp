<![CDATA[Alkacon OpenCms Add-On Module Paket(OAMP) ]]> Das Alkacon OpenCms Add-On Modul Paket (auch OAMP genannt) beinhaltet einen Satz von kostenfreien Erweiterungen für OpenCms. Alkacon OAMP bietet verschiedene Funktionen, welche zum Aufbau von interaktiven Webseiten genutzt werden können.

Die Alkacon OAMP Module werden als Open Source Software von der Alkacon Software GmbH entwickelt. Alkacon ist ebenfalls für die Entwicklung des Open Source Kerns von OpenCms federführend verantwortlich. Den vollständigen Text der OAMP Lizenzbestimmungen finden sie hier (in Englisch).

Die Alkacon OAMP Module werden auf dem standardmäßigen OpenCms System installiert. Der Source Code der OAMP Module ist im öffentlichen OpenCms CVS verfügbar. Die Module werden separat zum Download angeboten.
