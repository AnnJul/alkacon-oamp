<![CDATA[Alkacon OAMP Syndication Feed Module]]> fb0c0281-e78f-11dc-8533-3bdd2ea0b1ac f19afd7c-e792-11dc-8533-3bdd2ea0b1ac 51c96b2d-e78a-11dc-8533-3bdd2ea0b1ac Das Modul erweitert Ihre OpenCms Installation mit der Möglichkeit verschiedene Versionen von RSS-Feeds (wie RSS oder Atom) zu erstellen. Dies geschieht mit Hilfe von strukturierten XML Dateien (XmlContents). Diese Features werden zur Verfügung gestellt:

  • Erstellen von RSS feeds aus Xml Contents
  • Komplette Konfiguration eines Feeds mit mit nur einer Datei gemacht
  • Auswahl von vielen verschiedenen Typen von RSS-Feeds (z.B. RSS 0.9, RSS 1.0, RSS 2.0, Atom 0.3, Atom 1.0) welche zur Erstellung der Feeds verwendet werden können
  • Möglichkeit zur Angabe eines Bildes für den Feed
  • Verwendung der OpenCms Kollektoren um zu konfigurieren welche XML contents im Feed angezeigt werden sollen
  • konfigurierbare Zuweisung der einzelnen Elemente der XML contents zu den Feldern im Feed mit der Möglichkeit zur Angabe eines Standardwertes und einer maximalen Länge
