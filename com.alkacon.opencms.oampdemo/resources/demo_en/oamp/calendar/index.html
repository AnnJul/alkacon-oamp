<![CDATA[Alkacon OAMP Calendar Module]]> 871d6982-1533-11dd-9751-d323604c3b7a The module extends a basic OpenCms installation with the capability to display different
calendar based views on time based events. It provides the following features:

  • Different JSP side elements to be included in an existing template showing a monthly overview or the most recent events.
  • Daily, weekly, monthly and yearly overviews can be shown.
  • Easy configuration of resource types that should be displayed in the calendar views.
  • Shows local holidays like New Year's Day or Christmas.
  • Additionally, it is possible to define serial dates appearing more than once in the calendar.
